# Anti-Aliasing

## Temporal Anti-Aliasing (TAA)

Temporal anti-aliasing (TAA) technique that reduces aliasing by blending data from previous frames by using jittering.

### Sub-pixel Jitter

In ray tracing, this method is a form of TAA that slightly offsets a ray's origin. Combined with progressive rendering, the offsets allows the gathering and averaging of =different samples to remove aliasing
