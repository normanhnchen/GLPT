# Tone Mapping

Tone mapping is the process of using a mathematical curve to convert from high-range luminance values into the $[0, 1]$ range that a standard display expects. Also, it also can help produce an image which provokes realism or artistic choice.

In a tonemap curve, there usually includes the common features:

- The shoulder: maps high luminance values to the range $[0, 1]$
- The foot: controls dark display regions
